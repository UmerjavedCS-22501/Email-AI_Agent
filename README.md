# 📧 AI Email Assistant Agent

An **AI-powered Email Assistant Agent** that automatically reads unread emails from Gmail, generates short AI summaries, and sends those summaries directly to **Slack**.

This project demonstrates how **AI Agents built with LangChain** can automate real-world communication workflows by integrating with external services like Gmail and Slack.

---

## 🚀 Features

- Automatically fetches **unread emails from Gmail**
- Uses an **LLM to generate concise summaries**
- Sends summaries directly to a **Slack channel**
- Built using **LangChain Agents and Tools**
- Fully automated workflow
- Easily extendable with more tools
